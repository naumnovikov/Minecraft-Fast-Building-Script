# Minecraft-Fast-Building-Script

Этот скрипт на AutoHotkey предназначен для автоматизации некоторых действий в игре Minecraft, таких как: быстрое строительство прямого моста, быстрое строительство моста «в высоту», а также шестерное нажатие.



## Требования

- ОС Windows
- AutoHotkey v1.1



## Управление

### Строительство прямого моста

Нужно встать на последний блок, нацелиться на координату угла pitch = 81.9. Нажав на Home, персонаж начнет быстро строить мост. Для того, что прервать цикл, нужно нажать End.

### Строительство моста «в высоту»

Нужно зажать клавишу X. Как только клавиша отпускается, строительство прекращается. Для того, что прервать цикл, нужно нажать End.

### Шестерное нажатие

Чтобы активировать, необходимо нажать XButton1. Для отключения - XButton2. Для того, что прервать цикл, нужно нажать End.

Для полного отключения программы нажмите Del.
